# Modular CSS-HW1
## Description
Use the supplied CSS file as starting point, break this down and move parts of it into **modular SASS files**.
## Technologies
* Visual Studio Code
* Window powershell
* Git Bash
## Prerequisites
* HTML
* CSS
* SASS
## Code Example
Compile SASS into CSS file:`sass --watch sass:css --style compressed`
